This project investigates sensor-fusion and probabilistic SLAM algorithms for autonomous
underwater vehicles operating without GPS. The work combines acoustic positioning,
inertial measurement, and AI-based map estimation to enable reliable long-range navigation
in complex environments.

Key contributions:
- Particle-filter-based localisation under heavy noise
- Multi-vehicle cooperative mapping
- Integration with real-time path planning
